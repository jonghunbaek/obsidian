# 1차 풀이
```java
public int[] solution(String s) {  
    int[] results = new int[s.length()];  
    Map<Character, Integer> indexByChar = new HashMap<>();  
    for (int i = 0; i < s.length(); i++) {  
        char c = s.charAt(i);  
        Integer index = indexByChar.getOrDefault(c, -1);  
        results[i] = index == -1 ? -1 : i - index;  
  
        indexByChar.put(c, i);  
    }  
  
    return results;  
}
```

# 2차 풀이
```java
import java.util.*;

class Solution {
    public int[] solution(String s) {
        Map<Character, Integer> indexByChar = new HashMap<>();
        int[] results = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            char target = s.charAt(i);
            int idx = indexByChar.getOrDefault(target, -1);
            if (idx == -1) {
                results[i] = -1;
            } else {
                results[i] = i - idx;
            }
            indexByChar.put(target, i);
        }
        
        return results;
    }
}
```
