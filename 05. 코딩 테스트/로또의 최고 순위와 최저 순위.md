# 1차 풀이
```java
class Solution {
    public int[] solution(int[] lottos, int[] win_nums) {
        int[] ranking = new int[7];
        int value = 6;
        for (int i = 1; i < ranking.length; i++) {
            ranking[i] = value;
            value--;
        }
        
        int count = 0;
        int zeroCount = 0;
        for (int i = 0; i < lottos.length; i++) {
            if (lottos[i] == 0) {
                zeroCount++;
                continue;
            }
            
            for (int j = 0; j < win_nums.length; j++) {
                if (lottos[i] == win_nums[j]) count++;    
            }
        }
        
        int win = count + zeroCount > 6 ? 1 : ranking[count + zeroCount];
        int lose = count <= 1 ? 6 : ranking[count];
        
        return new int[] {win, lose};
    }
}
```
테케 1개가 틀렸는데, 도저히 뭔지 찾지를 못하겠음
- 완전히 멍청한 실수를 함
- ranking[0]의 값을 설정 안해줘서 그랬음
- 마지막 win, lose 값 계산시에 다 커버했다고 자만한게 원인

# 2차 풀이
```java
class Solution {
    public int[] solution(int[] lottos, int[] win_nums) {
        int[] ranking = new int[] {6, 6, 5, 4, 3, 2, 1};
        int zeroCount = 0;
        int matchCount = 0;
        for (int l : lottos) {
            if (l == 0) {
                zeroCount++;
                continue;
            }
            
            for (int number : win_nums) {
                if (number == l) matchCount++;
            }
        }
        
        return new int[] {ranking[matchCount + zeroCount], ranking[matchCount]};
    }
}
```
