# 1차 풀이
```java
// 구현, 스택
import java.util.*;

class Solution {
    public int solution(int[][] board, int[] moves) {
        Stack<Integer> basket = new Stack<>();
        int count = 0;
        for (int move : moves) {
            int col = move - 1;
            
            for (int i = 0; i < board.length; i++) {
                if (board[i][col] == 0) continue;
                
                if (!basket.isEmpty() && basket.peek() == board[i][col]) {
                    basket.pop();
                    count += 2;
                    board[i][col] = 0;
                    break;
                } 
                
                basket.push(board[i][col]);
                board[i][col] = 0;
                break;    
            }
            
            
        }
        
        return count;
    }
}
```
대부분의 테케를 틀려 왜 그런가 한참 고민
인형을 터뜨릴 때 `board[i][col] = 0;` 이 로직을 빼먹어서 생긴문제
