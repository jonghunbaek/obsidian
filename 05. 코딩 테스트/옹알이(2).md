# 1차 풀이
```java

class Solution {
    public int solution(String[] babbling) {
        String[] arr = new String[] {"aya", "ye", "woo", "ma"};
        int count = 0;
        for (int i = 0; i < babbling.length; i++) {
            babbling[i] = babbling[i].replaceAll(arr[0], "0");
            babbling[i] = babbling[i].replaceAll(arr[1], "1");
            babbling[i] = babbling[i].replaceAll(arr[2], "2");
            babbling[i] = babbling[i].replaceAll(arr[3], "3");

            boolean isSafe = !babbling[i].isBlank();
            char previous = ' ';
            for (char c : babbling[i].toCharArray()) {
                if (!Character.isDigit(c) || previous == c) isSafe = false;
                previous = c;
            }

            if (isSafe) count++;
        }

        return count;
    }
}
```

# 2차 풀이
```java
class Solution {
    public int solution(String[] babbling) {
        String[] conditions = {"aya", "ye", "woo", "ma"};
        int count = 0;
        for (String b : babbling) {
            for (int i = 0; i < conditions.length; i++) {
                b = b.replace(conditions[i], String.valueOf(i));
            }
            
            char previous = ' ';
            boolean isSafe = true;
            for (int i = 0; i < b.length(); i++) {
                if (!Character.isDigit(b.charAt(i))) {
                    isSafe = false;
                    break;
                }
                if (previous == b.charAt(i)) {
                    isSafe = false;
                    break;
                }
                
                previous = b.charAt(i);
            }
            
            if (isSafe) count++;
        }
        
        return count;
    }
}
```

# 3차 풀이
```java
class Solution {
    public int solution(String[] babbling) {
        int result = 0;
        for (String b : babbling) {
            b = b.replaceAll("ayaaya|yeye|wooowoo|mama", "1");
            b = b.replaceAll("aya|ye|woo|ma", "");
            if (b.isEmpty()) result++;
        }
        
        return result;
    }
}
```
