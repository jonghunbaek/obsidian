# 1차 풀이
```java
class Solution {
    public String solution(int[] food) {
        StringBuilder sb = new StringBuilder(); 
        for (int i = 1; i < food.length; i++) {
            int time = food[i] / 2;
            for (int j = 0; j < time; j++) {
                sb.append(i);
            }
        }
        
        return sb.toString() + "0" + sb.reverse().toString();
    }
}
```
