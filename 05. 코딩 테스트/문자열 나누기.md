# 1차 풀이
```java
import java.util.*;
class Solution {
    public int solution(String s) {
        Stack<Character> stack = new Stack<>();
        int equalCount = 0;
        int diffCount = 0;
        int count = 0;
        for (int i = 0; i < s.length(); i++) {
            if (stack.isEmpty()) {
                stack.push(s.charAt(i));
            }

            if (stack.peek() == s.charAt(i)) {
                equalCount++;
            } else {
                diffCount++;
            }

            if (diffCount == equalCount) {
                stack.clear();
                count++;
            }
        }

        return stack.isEmpty() ? count : count + 1;
    }
}
```
최초에 문자열관련 문제라 생각하고 진행하다가 계속 안풀려서 다시 생각해보니 스택을 사용해야 했음을 깨달음
문제 풀이 전략을 초기에 확실하게 세울것

# 2차 풀이
```java
import java.util.*;

class Solution {
    public int solution(String s) {
        Stack<Character> stack = new Stack<>();
        int count = 0;
        int diffCount = 0;
        for (char c : s.toCharArray()) {
            if (stack.isEmpty()) {
                stack.push(c);
                count++;
                continue;
            }
            
            if (stack.peek() == c) {
                stack.push(c);
                continue;
            }
            
            diffCount++;
            if (stack.size() == diffCount) {
                diffCount = 0;
                stack.clear();
            }
        }
        
        return count;
    }
}
```
스택을 이용한 풀이는 항상 직관적이이 않아서 그런지 풀이에 시간이 꽤걸림

# 3차 풀이
```java
import java.util.*;

class Solution {
    public int solution(String s) {
        Stack<Character> stack = new Stack<>();
        int count = 0;
        int diffCount = 0;
        for (char c : s.toCharArray()) {
            if (stack.isEmpty()) {
                stack.push(c);
                count++;
                continue;
            }
            
            if (!stack.isEmpty() && stack.peek() == c) {
                stack.push(c);
                continue;
            }
            
            diffCount++;
            
            if (stack.size() == diffCount) {
                stack.clear();
                diffCount = 0;
            }
        }
        
        return count;
    }
}
```
