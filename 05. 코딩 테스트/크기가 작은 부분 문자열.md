# 1차 풀이
```java
class Solution {
    public int solution(String t, String p) {
        int size = p.length();
        long origin = Long.parseLong(p);
        int idx = 0;
        int answer = 0;
        while (idx <= t.length() - size) {
            long target = Long.parseLong(t.substring(idx, idx + size));
            if (target <= origin) answer++;
            idx++;
        }
        return answer;
    }
}
```

# 2차 풀이
```java
class Solution {
    public int solution(String t, String p) {
        int count = 0;
        long origin = Long.parseLong(p);
        for (int i = 0; i < t.length() - p.length() + 1; i++) {
            long target = Long.parseLong(t.substring(i, i + p.length()));
            if (target <= origin) count++;
        }
        
        return count;
    }
}
```
