# 1차 풀이
```java
public int solution(int[] ingredient) {  
    int[] hamburger = new int[] {1, 2, 3, 1};  
    int count = 0;  
    Stack<Integer> stack = new Stack<>();  
    for (int target : ingredient) {  
        if (stack.isEmpty() && target != 1) {  
            continue;  
        }  
        if (target == 1) {  
            stack.push(target);  
            continue;  
        }  
  
        if (stack.peek() == 1 && target == 2) {  
            stack.push(target);  
        } else if (stack.peek() == 2 && target == 3) {  
            stack.push(target);  
        } else if (stack.peek() == 3 && target == 1) {  
            for (int j = 0; j < 3; j++) {  
                stack.pop();  
            }  
  
            count++;  
        }  
    }  
  
    int idx = 3;  
    int answerCount = 0;  
    while (!stack.isEmpty()) {  
        int value = stack.pop();  
        if (value == hamburger[idx]) {  
            answerCount++;  
        }  
  
        if (answerCount == hamburger.length) {  
            count++;  
            answerCount = 0;  
        }  
  
        idx = idx - 1 < 0 ? 3 : idx - 1;  
    }  
  
    return count;  
}
```

# 2차 풀이
```java
import java.util.*;

class Solution {
    public int solution(int[] ingredient) {
        Stack<Integer> stack = new Stack<>();
        int result = 0;
        for (int i : ingredient) {
            stack.push(i);
            
            if (stack.size() >= 4) {
                int size = stack.size();
                if (stack.get(size - 1) == 1 && stack.get(size - 2) == 3 && stack.get(size - 3) == 2 && stack.get(size - 4) == 1) {
                    result++;
                    stack.pop();
                    stack.pop();
                    stack.pop();
                    stack.pop();
                }
            }
        }
        
        return result;
    }
}
```

아래와 같이 배열을 덮어 쓰는 방법으로도 해결 가능
```java
class Solution {
    public int solution(int[] ingredient) {
        int[] stack = new int[ingredient.length];
        int sp = 0;
        int answer = 0;
        for (int i : ingredient) {
            stack[sp++] = i;
            if (sp >= 4 && stack[sp - 1] == 1
                && stack[sp - 2] == 3
                && stack[sp - 3] == 2
                && stack[sp - 4] == 1) {
                sp -= 4;
                answer++;
            }
        }
        return answer;
    }
}
```
