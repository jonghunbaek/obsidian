# 1차 풀이
```java
// 재귀
import java.util.*;

class Solution {
    public int solution(int[] number) {
        return count(0, 0, number, new int[3]);
    }
    
    private int count(int depth, int idx, int[] number, int[] results) {
        if (depth == 3) {
            int sum = Arrays.stream(results)
                .sum();
        
            if (sum == 0) {
                return 1;
            }
            
            return 0;
        }
        
        int totalSum = 0;
        for (int i = idx; i < number.length; i++) {
            results[depth] = number[i];
            totalSum += count(depth + 1, i + 1, number, results);
            results[depth] = 0;
        }
        
        return totalSum;
    }
}
```
3중 for문으로 풀어도 이상 없지만 연습삼아 재귀로 풀이
`results[depth] = 0;` 를 주석 처리해도 통과됨

# 2차 풀이
```java
class Solution {
    public int solution(int[] number) {
        return count(0, 0, 0, number);
    }
    
    private int count(int depth, int idx, int sum, int[] number) {
        if (depth == 3) {
            if (sum == 0) return 1;
            return 0;
        }
        
        int result = 0;
        for (int i = idx; i < number.length; i++) {
            result += count(depth + 1, i + 1, sum + number[i], number);
        }
        
        return result;
    }
}
```
