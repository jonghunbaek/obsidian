# 1차 풀이
```java
class Solution {
    public int solution(int n) {
        long[] dp = new long[100001];
        dp[0] = 0;
        dp[1] = 1;
        for (int i = 2; i <= 100000; i++) {
            dp[i] = (dp[i - 1] + dp[i - 2]) % 1234567;
        }
        
        return (int) dp[n];
    }
}
```
피보나치 수로 더해가다보면 n이 92가 넘을 경우 long타입도 오버플로우가 발생함
미리 모듈러 연산을 통해 값을 줄여야함
