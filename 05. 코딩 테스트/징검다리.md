# 1차 풀이
```java
import java.util.Arrays;

public class Solution {
    private boolean isValid(int d, int[] rocks, int n) {
        int removed = 0;  // 제거한 바위의 개수
        int last = 0;  // 마지막 바위의 위치
        for (int rock : rocks) {
            if (rock - last < d) {
                removed++;
                continue;
            }

            last = rock;
        }
        return removed <= n;
    }

    public int solution(int distance, int[] rocks, int n) {
        rocks = Arrays.copyOf(rocks, rocks.length + 1);
        rocks[rocks.length - 1] = distance;
        Arrays.sort(rocks);

        int start = 1;
        int end = distance + 1;

        while (end - start > 1) {
            int d = (start + end) / 2;

            if (isValid(d, rocks, n)) {
                start = d;
            } else {
                end = d;
            }
        }

        return start;
    }
}
```
해당 코드는 정답을 그대로 옮긴 풀이. 
설명을 봐도 정확히 이해가 안됨

# 2차 풀이
```java
class Solution {
    public int solution(int distance, int[] rocks, int n) {
        rocks = Arrays.copyOf(rocks, rocks.length + 1);
        rocks[rocks.length - 1] = distance;
        Arrays.sort(rocks);
        
        int start = 1;
        int end = distance + 1;
        
        while (end - start > 1) {
            int mid = (start + end) / 2;
            
            if (isSmallest(mid, rocks, n)) {
                end = mid;
            } else {
                start = mid;
            }
        }
        
        return start;
    }
    
    private boolean isSmallest(int mid, int[] rocks, int n) {
        int count = 0;
        int last = 0;
        for (int rock : rocks) {
            if (rock - last < mid) {
                count++;
                continue;
            }
            
            last = rock;
        }
        
        return count > n;
    }
}
```
일단 해당 문제가 정답을 만족하는 값 중, 최댓값을 찾는 문제라는 것은 이해했다.
하지만 여전히 이해가 가지 않는 부분은 while문..
