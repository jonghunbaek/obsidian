# 1차 풀이
```java
import java.util.*;
class Solution {
    public int solution(String s) {
        Stack<Character> stack = new Stack<>();
        int equalCount = 0;
        int diffCount = 0;
        int count = 0;
        for (int i = 0; i < s.length(); i++) {
            if (stack.isEmpty()) {
                stack.push(s.charAt(i));
            }

            if (stack.peek() == s.charAt(i)) {
                equalCount++;
            } else {
                diffCount++;
            }

            if (diffCount == equalCount) {
                stack.clear();
                count++;
            }
        }

        return stack.isEmpty() ? count : count + 1;
    }
}
```
