# 1차 풀이
```java
import java.util.*;

class Solution {
    public int solution(int n, int[][] results) {
        List<List<Integer>> winGraph = new ArrayList<>(); // 내가 이기는 선수들
        List<List<Integer>> loseGraph = new ArrayList<>(); // 내가 지는 선수들

        for (int i = 0; i <= n; i++) {
            winGraph.add(new ArrayList<>());
            loseGraph.add(new ArrayList<>());
        }
        
        for (int[] result : results) {
            int winner = result[0];
            int loser = result[1];
            winGraph.get(winner).add(loser);
            loseGraph.get(loser).add(winner);
        }
        
        int answer = 0;
        for (int i = 1; i <= n; i++) {
            if (count(i, n, winGraph) + count(i, n, loseGraph) == n - 1) {
                answer++;
            }
        }
        
        return answer;
    }
    
    private int count(int startNode, int n, List<List<Integer>> graph) {
        Queue<Integer> q = new LinkedList<>();
        boolean[] visited = new boolean[n + 1];
        q.add(startNode);
        visited[startNode] = true;
        
        int count = 0;
        while(!q.isEmpty()) {
            int current = q.poll();
            for (int next : graph.get(current)) {
                if (visited[next]) {
                    continue;
                }
                
                visited[next] = true;
                q.add(next);
                count++;
            }
        }
        
        return count;
    }
}
```
풀이법 자체는 단순함. dfs, bfs 모두 가능.
근데 dfs로 풀라하면 못풀것 같음.
2차원 배열을 사용해 이기고 진 경우를 미리 기록해두는 것이 핵심

# 2차 풀이
```java
class Solution {
    public int solution(int n, int[][] results) {
        boolean[][] winners = new boolean[n + 1][n + 1];
        boolean[][] losers = new boolean[n + 1][n + 1];
        for (int[] result : results) {
            int winner = result[0];
            int loser = result[1];
            winners[winner][loser] = true;
            losers[loser][winner] = true;
        }
        
        boolean[] visited = new boolean[n + 1];
        int answer = 0;
        for (int player = 1; player <= n; player++) {
            int winnerCount = dfs(player, visited, winners);
            visited = new boolean[n + 1];
            
            int loserCount = dfs(player, visited, losers);
            visited = new boolean[n + 1];
            
            if (winnerCount + loserCount == n - 1) {
                answer++;
            }
        }
        
        return answer;
    }
    
    private int dfs(int current, boolean[] visited, boolean[][] results) {
        int count = 0;
        for (int i = 1; i < results[current].length; i++) {
            if (visited[i] || !results[current][i]) {
                continue;
            }
            
            visited[i] = true;
            count += 1 + dfs(i, visited, results);
        }
        
        return count;
    }
}
```
하나의 그래프로도 처리할 수 있음.
승자, 패자를 담는 2차원 배열과 다르게 방문 배열은 1차원 배열임에 주의 필요
-> 방문 배열이 1차원인 것은 해당 행 전체를 탐색할 필요가 없기 때문임

# 3차 풀이
```java
class Solution {
    public int solution(int n, int[][] results) {
        boolean[][] winners = new boolean[n][n];
        boolean[][] losers = new boolean[n][n];
        for (int[] result : results) {
            winners[result[0] - 1][result[1] - 1] = true;
            losers[result[1] - 1][result[0] - 1] = true;
        }
        
        int answer = 0;
        boolean[] visited = new boolean[n];
        for (int i = 0; i < n; i++) {
            int win = count(winners, i, 0, visited);
            visited = new boolean[n];
            
            int lose = count(losers, i, 0, visited);
            visited = new boolean[n];
            
            if (win + lose + 1 == n) {
                answer++;
            }
        }
        
        return answer;
    }
    
    private int count(boolean[][] map, int start, int value, boolean[] visited) {
        for (int i = 0; i < map.length; i++) {
            if (visited[i]) {
                continue;
            }
            
            if (map[start][i]) {
                visited[i] = true;
                value = count(map, i, value + 1, visited);
            }
        }
        
        return value;
    }
}
```
1. 방문 배열 미사용으로 인해 틀림
2. count 재귀에서 값을 계산하는 방식도 틀림
3. 값 계산 방식은 인자로 넘기는 방식, 직접 계산해서 반환하는 방식 크게 두 가지가 있음

class Solution {

        boolean[][] wins = new boolean[n][n];

        for (int[] result : results) {

            int loseCount = dfs(i, loses, visited);
            visited = new boolean[n];
            
            if (winCount + loseCount + 1 == n) {
                answer++;
            }
        }
        
        return answer;
    }
    
    private int dfs(int no, boolean[][] result, boolean[] visited) {
        int count = 0;
        for (int i = 0; i < result.length; i++) {
            if (visited[i] || !result[no][i]) {
                continue;
            }
            
            visited[i] = true;
            count += dfs(i, result, visited) + 1;
        }
        
        return count;
    }
}

# 6차 풀이
```java
class Solution {
    public int solution(int n, int[][] results) {
        int[][] wins = new int[n][n];
        int[][] loses = new int[n][n];
        
        for (int[] result : results) {
            int winner = result[0] - 1;
            int loser = result[1] - 1;
            wins[winner][loser] = 1;
            loses[loser][winner] = 1;
        }
        
        boolean[] visited = new boolean[n];
        int result = 0;
        for (int i = 0; i < n; i++) {
            int winCount = count(i, wins, visited);
            visited = new boolean[n];
            int loseCount = count(i, loses, visited);

            if (winCount + loseCount + 1 == n) {
                result++;
            }
        }
        return result;
    }
    
    private int count(int start, int[][] result, boolean[] visited) {
        int count = 0;
        for (int i = 0; i < result.length; i++) {
            if (visited[i] || result[start][i] == 0) {
                continue;
            }
            
            visited[i] = true;
            count += count(i, result, visited) + 1;
        }
        
        return count;
    }
}
```
