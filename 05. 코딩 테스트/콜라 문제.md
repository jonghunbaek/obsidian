# 1차 풀이
```java
class Solution {
    public int solution(int a, int b, int n) {
        int sum = 0;
        while (n >= a) {
            sum += (n / a) * b;
            n = (n / a) * b + (n % a);
        }
        
        return sum;
    }
}
```
