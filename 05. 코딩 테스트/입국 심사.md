# 1차 풀이
```java
class Solution {
    public long solution(int n, int[] times) {
        long start = 1;
        long end = 1000000000000000000L;
        
        while (end > start) {
            long middle = (start + end) / 2;
            
            if (isShorter(middle, n, times)) {
                end = middle;                            
            } else {
                start = middle + 1;
            }
        }
        
        return start;
    }
    
    private boolean isShorter(long target, int n, int[] times) {
        long temp = 0;
        for (int time : times) {
            temp += target / time;
        }
        
        return temp >= n;
    }
}
```
