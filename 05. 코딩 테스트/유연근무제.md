# 1차 풀이
```java
// 1. 단순 구현
// 2. 주말은 빼야함에 유의
// 3. timelogs의 시작 요일은 startday에 따라 달라짐

// 0 1 2 3 4 5 6
// 1 2 3 4 5 6 7
// 1 +5
// 2 +4
// 3 +3
// 4 +2
// 5 +1
// 6 0
// 7 -1
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        for (int i = 0; i < schedules.length; i++) {
            schedules[i] += 10;
        }
        
        int count = 0;
        for (int k = 0; k < timelogs.length; k++) {
            int saturday = 0;
            int sunday = 0;
            if (startday < 6) {
                saturday = (6 - startday) % 7;
                sunday = (6 - startday) % 7 + 1;
            } else {
                saturday = startday == 6 ? 0 : 6;
                sunday = startday == 6 ? 1 : 0;
            }
             
            boolean isSafe = true;
            for (int i = 0; i < timelogs[k].length; i++) {
                if (i == saturday || i == sunday) continue;
                if (timelogs[k][i] > schedules[k]) {
                    isSafe = false;
                    break;
                }
            }
            
            if (isSafe) count++;
        }
        
        return count;
    }
}
```
사실상 다 맞췄지만 schedule의 원소를 +10해주는 부분에서 틀림.
또 토,일요일을 구하는 계산식이 매끄럽지 않음.
