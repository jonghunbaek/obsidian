# 1차 풀이
```java
// 1. 소수 구해서 divisors에 2로 저장
// 2. divisors의 값이 2가 아니라면 약수 계산

class Solution {
    public int solution(int number, int limit, int power) {
        int result = 0;
        for (int i = 1; i <= number; i++) {
            int count = getDivisorCount(i);
            
            if (count > limit) {
                result += power;
            } else {
                result += count;
            }
        }
        
        return result;
    }
    
    private int getDivisorCount(int target) {
        int result = 0;
        for(int i = 1; i <= Math.sqrt(target); i++) {
            if (i == Math.sqrt(target)) {
                result++;
            } else if (target % i == 0) {
                result += 2;
            }
        }
        
        return result;
    }
}
```
약수를 효율적으로 구할 방법이 떠오르지 않아 틀림
