# 1차 풀이
```java
// 완전탐색
import java.util.*;

class Solution {
    public int solution(int[] d, int budget) {
        Arrays.sort(d);

        int sum = 0;
        for (int i = 0; i < d.length; i++) {
            sum += d[i];
            if (sum > budget) {
                return i;
            }
        }

        return d.length;
    }
}
```

처음에 완전 탐색이라 생각하고 재귀로 접근해 풀이에 한참 걸림
```java
public int solution(int[] d, int budget) {  
    return plan(0, 0, 0, d, budget);  
}  
  
private int plan(int idx, int sum, int count, int[] d, int budget) {  
    if (sum > budget) {  
        return count - 1;  
    }  
  
    if (count == d.length || idx == d.length) {  
        return count;  
    }  
  
    int max = 0;  
    for (int i = idx; i < d.length; i++) {  
        int value = plan(i + 1, sum + d[i], count + 1, d, budget);  
        max = Math.max(max, value);  
        if (max == d.length) {  
            break;  
        }  
    }  
  
    return max;  
}
```

# 2차 풀이
```java
import java.util.*;

class Solution {
    public int solution(int[] d, int budget) {
        Arrays.sort(d);
        for (int i = 0; i < d.length; i++) {
            budget -= d[i];
            if (budget < 0) return i;
        }
        
        return d.length;
    }
}
```
