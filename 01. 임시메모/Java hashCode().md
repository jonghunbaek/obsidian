### 원문 (인용) 
- 해시 인덱스를 사용하기 위해선 '해시 코드'가 필요하다. 
- Java에는 원시 타입, 참조 타입 등 다양한 타입이 존재한다.
- 모든 타입을 해시 자료 구조에 저장하려면 모든 객체가 숫자 해시 코드를 제공할 수 있어야 한다.
- Java는 Object에 있는 hashCode()를 통해 이를 구현한다.
	- 단, 해당 메서드를 그대로 사용하기 보단 오버라이딩해서 사용한다. 
	- 기본 구현은 객체의 참조 값을 기반으로 해시 코드를 생성한다.
	- 그렇기 때문에 객체의 인스턴스가 다르면 해시 코드도 다르게 된다.
