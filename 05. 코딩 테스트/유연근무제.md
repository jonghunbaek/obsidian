# 1차 풀이
```java
// 1. 단순 구현
// 2. 주말은 빼야함에 유의
// 3. timelogs의 시작 요일은 startday에 따라 달라짐

// 0 1 2 3 4 5 6
// 1 2 3 4 5 6 7
// 1 +5
// 2 +4
// 3 +3
// 4 +2
// 5 +1
// 6 0
// 7 -1
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        for (int i = 0; i < schedules.length; i++) {
            schedules[i] += 10;
        }
        
        int count = 0;
        for (int k = 0; k < timelogs.length; k++) {
            int saturday = 0;
            int sunday = 0;
            if (startday < 6) {
                saturday = (6 - startday) % 7;
                sunday = (6 - startday) % 7 + 1;
            } else {
                saturday = startday == 6 ? 0 : 6;
                sunday = startday == 6 ? 1 : 0;
            }
             
            boolean isSafe = true;
            for (int i = 0; i < timelogs[k].length; i++) {
                if (i == saturday || i == sunday) continue;
                if (timelogs[k][i] > schedules[k]) {
                    isSafe = false;
                    break;
                }
            }
            
            if (isSafe) count++;
        }
        
        return count;
    }
}
```
사실상 다 맞췄지만 schedule의 원소를 +10해주는 부분에서 틀림.
또 토,일요일을 구하는 계산식이 매끄럽지 않음.
# 2차 풀이

```java
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        for (int i = 0; i < schedules.length; i++) {
            schedules[i] = addExtraMinutes(schedules[i]);
        }
        
        int result = 0;
        for (int num = 0; num < timelogs.length; num++) {
            int saturday = (7 - startday) == 0 ? 7 : 7 - startday;
            int sunday = (saturday + 1) > 7 ? 1 : saturday + 1;
            
            boolean isSafe = true;
            for (int i = 0; i < timelogs[num].length; i++) {
                if (i + 1 == saturday || i + 1 == sunday) continue;
                if (timelogs[num][i] > schedules[num]) isSafe = false;
            }
            
            if (isSafe) result++;
        }
        
        return result;
    }
    
    private int addExtraMinutes(int originTime) {
        int hour = originTime / 100;
        int min = originTime % 100;
        
        if (min + 10 <= 59) return hour * 100 + (min + 10);
        
        return (hour + 1) * 100 + (min + 10 - 60);
    }
}
```

# 3차 풀이
```java
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        for (int i = 0; i < schedules.length; i++) {
            schedules[i] = addBufferTime(schedules[i]);
        }
        
        int result = 0;
        for (int i = 0; i < timelogs.length; i++) {
            boolean isSafe = true;
            int saturday = 6 - startday < 0 ? 6 : 6 - startday;
            int sunday = saturday + 1 > 6 ? 0 : saturday + 1;
            for (int j = 0; j < timelogs[i].length; j++) {
                if (j == saturday || j == sunday) continue;
                
                if (timelogs[i][j] > schedules[i]) {
                    isSafe = false;
                    break;
                }
            }
            
            if (isSafe) result++;
        }
        
        return result;
    }
    
    private int addBufferTime(int target) {
        int bufferTime = 10;
        int hour = target / 100;
        int minutes = target % 100;
        
        return minutes + bufferTime >= 60 ? (hour + 1) * 100 + 10 - (60 - minutes) : hour * 100 + (minutes + 10);
    }
}
```

0-based : int dayOfWeek = (startday + j) % 7;
1-based :  int dayOfWeek = (startday + j - 1) % 7 + 1;
