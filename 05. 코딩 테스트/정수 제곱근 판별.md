# 1차 풀이
```java
class Solution {
    public long solution(long n) {
        double value = Math.sqrt(n);
        if (Math.ceil(value) != value) return -1;
        long result = (long) value + 1;
        return result * result;
    }
}
```
제곱구하는 메서드 사용법 숙지
```java
class Solution {
  public long solution(long n) {

    double i = Math.sqrt(n);

    return Math.floor(i) == i ? (long) Math.pow(i + 1, 2) : -1;
  }
}
```
