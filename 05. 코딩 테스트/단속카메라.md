# 1차 풀이
```java
import java.util.*;

class Solution {
    public int solution(int[][] routes) {
        Arrays.sort(routes, Comparator.comparingInt(a -> a[1]));

        int result = 0;
        int last = -30001;
        for (int[] route : routes) {
            if (route[0] <= last && route[1] >= last) {
                continue;
            }

            result++;
            last = route[1];
        }

        return result;
    }
}
```

# 2차 풀이
```java
import java.util.*;

// -20          -15
//      -18             -13
//                  -14         -5
//                              -5      -3
class Solution {
    public int solution(int[][] routes) {
        Arrays.sort(routes, (r1, r2) -> r1[1] - r2[1]); // 나간 지점 오름차순 정렬
        
        int count = 0;
        int lastCamera = -30001; // 최소값이 30,000
        for(int[] route : routes) {
            int in = route[0];
            int out = route[1];
            
            if (lastCamera >= in && lastCamera <= out) {
                continue;
            }
            
            count++;
            lastCamera = out;
        }
        
        return count;
    }
}
```
