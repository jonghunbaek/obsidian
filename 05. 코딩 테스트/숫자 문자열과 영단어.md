# 1차 풀이
```java
import java.util.*;

class Solution {
    
    private static final String[] NUMBERS = {"zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"};

    public int solution(String s) {
        StringBuilder sb = new StringBuilder();
        int idx = 0;
        while(idx < s.length()) {
            char target = s.charAt(idx);
            if (Character.isDigit(target)) {
                sb.append(target);
                idx++;
                continue;
            }

            String word = s.substring(idx, idx + 2);
            for (int i = 0; i < NUMBERS.length; i++) {
                if (NUMBERS[i].startsWith(word)) {
                    sb.append(i);
                    idx += NUMBERS[i].length();
                    break;
                }
            }
        }

         return Integer.parseInt(sb.toString());
    }
}
```
어렵지 않게 풀었지만 이게 맞나 싶을 정도로 찜찜함

# 2차 풀이
```java
import java.util.*;

class Solution {
    private static final String[] NUMBERS = {"zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"};
    
    public int solution(String s) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < s.length(); i++) {
            if (Character.isDigit(s.charAt(i))) {
                sb.append(s.charAt(i));
                continue;
            }
            
            for (int j = 0; j < NUMBERS.length; j++) {
                String number = NUMBERS[j];
                if (number.startsWith(s.substring(i, i + 2))) {
                    sb.append(j);
                    i += number.length() - 1;
                    break;
                }
            }
        }
        return Integer.parseInt(sb.toString());
    }
}
```
결국 지난 번과 같은 풀이.
다른 풀이 방식을 참고 해보자

```java
public class Solution {
    private static final String[] words = {
            "zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine",
    };

    public int solution(String s) {
        for (int i = 0; i < words.length; i++) {
            s = s.replace(words[i], Integer.toString(i));
        }

        return Integer.parseInt(s);
    }
}
```
음.. 처음에 잠깐 생각해봤을 땐 if문 처리를 많이해야 할거라 예상했음

# 3차 풀이
```java
class Solution {
    private String[] numbers = new String[] {
        "zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"
    };
    
    public int solution(String s) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            if (Character.isDigit(c)) {
                sb.append(c);
                continue;
            }
            
            String target = s.substring(i, i + 2);
            for (int j = 0; j < numbers.length; j++) {
                if (!numbers[j].startsWith(target)) continue;
                sb.append(j);
                i += numbers[j].length() - 1;
            }
        }
        
        return Integer.parseInt(sb.toString());
    }
}
```
