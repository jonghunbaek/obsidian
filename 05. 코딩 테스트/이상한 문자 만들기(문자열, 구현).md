# 1차 풀이
틀린 풀이
```java
class Solution {
    public String solution(String s) {
        String[] words = s.split(" ");
        
        StringBuilder result = new StringBuilder();
        for (String word : words) {
            System.out.println("word = " + word);
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < word.length(); i++) {
                if (!Character.isAlphabetic(word.charAt(i))) {
                    continue; 
                }
                    
                if (i % 2 == 0) {
                    sb.append(Character.toUpperCase(word.charAt(i)));
                } else {
                    sb.append(Character.toLowerCase(word.charAt(i)));
                }
            }
            
            result.append(sb.toString())
                .append(" ");
        }
        
        return result.toString()
            .substring(0, result.length() - 1);
    }
}
```

정답 풀이
```java
class Solution {
    public String solution(String s) {
        int idx = 0;
        StringBuilder result = new StringBuilder();
        for (char c : s.toCharArray()) {
            if (!Character.isAlphabetic(c)) {
                result.append(c);
                idx = 0;
                continue;
            }
            
            if (idx % 2 == 0) {
                result.append(Character.toUpperCase(c));
            } else {
                result.append(Character.toLowerCase(c));
            }
            
            idx++;
        }
        
        return result.toString();
    }
}
```

오랜만에 이 문제를 풀면서 공백을 split으로 분리했다. 분리한만큼 공백이 사라지기 때문에 똑같이 채워 넣어 줘야 하는데 이 부분을 애매하게 처리해 문제가 생겼다.
다시 예전 풀이를 보며 하나씩 순차적으로 넣어주는 것이 더 간편하다는 판단이 생겨 그대로 진행해 통과함.

# 2차 풀이

```java
class Solution {
    public String solution(String s) {
        int idx = 0;
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < s.length(); i++) {
            if (Character.isWhitespace(s.charAt(i))) {
                sb.append(s.charAt(i));
                idx = 0;
                continue;
            }
            
            if (idx % 2 == 0) {
                sb.append(Character.toUpperCase(s.charAt(i)));
            } else {
                sb.append(Character.toLowerCase(s.charAt(i)));
            }
            
            idx++;
        }
        
        return sb.toString();
    }
}
```

# 3차 풀이
```java
class Solution {
    public String solution(String s) {
        char[] results = new char[s.length()];
        int count = 0;
        for (int i = 0; i < s.length(); i++) {
            char target = s.charAt(i);
            if (Character.isWhitespace(target)) {
                count = 0;
                results[i] = ' ';
                continue;
            }

            if (count % 2 == 0) {
                results[i] = Character.toUpperCase(target);
            } else {
                results[i] = Character.toLowerCase(target);
            }

            count++;
        }

        return String.valueOf(results);
    }
}
```
